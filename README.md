GNU编译器集合（GCC）是由GNU项目生成的编译器系统，支持各种编程语言。 GCC是GNU工具链的关键组件。 自由软件基金会（FSF）根据GNU通用公共许可证（GNU GPL）分发GCC。 GCC在自由软件的发展中起着重要的作用，既是一个工具，也是一个例子。

属于编程语言类别，可以支持 C 语言开发。
